# Security Best Practices

## Code Review Guidelines

Always review code for the following security issues:
- SQL injection vulnerabilities
- Cross-site scripting (XSS)
- Hardcoded credentials
- Insecure cryptography

## Recommendations

When reviewing code, suggest secure alternatives and explain the risks.
Focus on practical, actionable advice that developers can implement immediately.

## Additional Resources

Refer to OWASP Top 10 and CWE Top 25 for authoritative guidance.
